# 🎮 Game Reviews Analysis

**Проект:** Анализ 10 000 отзывов на мобильные игры (Subway Surfers, PUBG, Candy Crush, Clash of Clans)  
**Автор:** GlebShalunkov  
**Технологии:** Python, Pandas, Matplotlib, Seaborn

## 📌 О проекте
Разведочный анализ, временные паттерны, анализ тональности отзывов.

## 🎯 Что сделано
- Очистка и обработка данных
- Анализ активности по часам и дням недели
- Простой анализ тональности (позитив/негатив/нейтрально)
- Визуализация ключевых метрик

## 🔥 Ключевые инсайты
- Средняя оценка: 4.43/5
- 77% отзывов — пятёрки
- Пик активности: воскресенье, 14:00
- Позитивных отзывов: 55%, негативных: 2.4%

## 📁 Файлы
- `game_reviews_analysis.ipynb` — основной ноутбук
- `game_reviews.csv` — исходные данные
- `game_stats.csv` — обработанная статистика
