# 🎮 Game Reviews Analysis

**Проект:** Анализ 10 000 отзывов на мобильные игры из Google Play  
**Дата:** 05.03.2026  
**Автор:** GLEBowski  
**Технологии:** Python, Pandas, Matplotlib, Seaborn, Jupyter

## 📊 Содержание проекта

| Файл | Описание |
|------|----------|
| game_reviews_analysis_full.xlsx | Полный отчёт с несколькими листами |
| total_stats.csv | Общая статистика по всем отзывам |
| game_stats.csv | Детальная статистика по каждой игре |
| hourly_stats.csv | Распределение отзывов по часам |
| daily_stats.csv | Распределение отзывов по дням недели |
| sentiment_stats.csv | Анализ тональности по играм |
